# 极智云表单引擎
极智云表单引擎，拖拉拽方式搭建业务表单，所见即所得。支持基本字段，用户、部门组织架构字段，图片、附件、富文本字段，数据选择、子表单字段20余种。为快速搭建业务应用提供强劲动力。<br/>
支持公式：实现业务计算规则可配置化。<br/>
支持数据联动：关联数据，选择数据，填充数据。实现各业务表单数据关联<br/>
